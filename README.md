# Rugmi

Rugmi is a simple client for [imgur.com](https://imgur.com/). 

## Project setup

1. Install dependencies:

```
npm install
```

2. Compiles and hot-reloads for development
```
npm run serve
```

3. Compiles and minifies for production

```
npm run build
```

4. Lints and fixes files

```
npm run lint
```
